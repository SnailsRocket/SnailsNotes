## 树

### 平衡二叉树

又被称为AVL树(有区别于AVL算法),

定义： 它是一颗空树，或者左右两个子树的高度差的绝对值不超过2



插入、删除都会破坏平衡性，这个时候我们需要进行平衡性的调整。



平衡性调整  单旋转  双旋转



### 红黑树(根节点必须为黑色)

红黑树的左右子树高度差有可能大于1

下面贴一个非常牛逼的网站

红黑树 自平衡的判断就是 左右子树的黑节点一样多
