## Docker常见命令

### 查看docker 命令

docker version

docker info



### 查看运行的容器

docker ps



### 查看docker中所有的容器

docker ps -a



### 查看images

 docker images  (-a)



### 启动、关闭、重启容器

docker start/stop/restart container_id



### 拉取镜像

docker pull mysql/redis/mongo



### 进入到容器中

docker exec 容器id



### 删除容器 镜像

docker rm container_id

docker rmi iamge_id



### Docker 命令API

docker --help

docker rm/exec/run/start/ps --help
