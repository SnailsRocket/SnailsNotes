## Docker常见命令

### 查看docker 命令

docker version

docker info





### 查看运行的容器

docker ps



### 查看docker中所有的容器

docker ps -a



### 查看images

 docker images  (-a)



### 启动、关闭、重启容器

docker start/stop/restart container_id
