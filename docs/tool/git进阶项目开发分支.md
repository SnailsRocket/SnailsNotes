## git 项目开发分支

一般我们使用git进行版本控制，主分支是master，然后还有Release、Develop、Feature分支

> master  主分支
>
> Release  不允许自我Review 并Merge代码
>
> Develop  项目开发分支，这个上面的版本，不能有bug
>
> Feature  这个分支到 Develop分支时，进行code review
