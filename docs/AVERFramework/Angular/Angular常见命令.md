##  Angular常见命令

##### ng new projectname

> 创建项目



##### ng serve /ng  -s

> start 前端项目



##### ng generate component  / ng g component

> 创建一个component



##### ng generate  interface

> 创建一个接口



##### ng version

> 查看版本



##### ng build

> 打包编译
>
> \# 这是生产构建
> ng build --target=production --environment=prod
> ng build --prod --env=prod
> ng build --prod
>
> 
>
> \# 这是开发构建
> ng build --target=development --environment=dev
> ng build --dev --e=dev
> ng build --dev
> ng build
